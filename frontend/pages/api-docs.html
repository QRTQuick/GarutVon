API Documentation

Integrate GarutVON's high-fidelity conversion cluster directly into your microservices. Authenticate securely with low latency pipelines.

Authentication

All Cloud API requests require passing your developer key in the X-API-Key custom header or via Bearer structure in Authorization header:

X-API-Key: gv_your_secret_key_here
Authorization: Bearer gv_your_secret_key_here

POST /api/v1/convert/image

Converts any supported input image file directly and transfers the converted output stream in the response body.

Request Parameters (Multipart/form-data)

Parameter Type Description
file Binary File The raw source image to convert (Max size 16MB).
format String Target output format extension (PNG, JPG, WEBP, BMP, TIFF, ICO).

Code Implementation Examples

Python (requests)

import requests

url = "http://localhost:8000/api/v1/convert/image"
headers = {"X-API-Key": "gv_your_api_key"}
files = {"file": open("photo.png", "rb")}
data = {"format": "WEBP"}

response = requests.post(url, headers=headers, files=files, data=data)
if response.status_code == 200:
    with open("converted.webp", "wb") as f:
        f.write(response.content)
    print("Success!")

cURL Command

curl -X POST "http://localhost:8000/api/v1/convert/image" \
  -H "X-API-Key: gv_your_api_key" \
  -F "file=@/path/to/photo.png" \
  -F "format=WEBP" \
  -o converted.webp

Node.js (Fetch)

const fs = require('fs');

const formData = new FormData();
formData.append('file', new Blob([fs.readFileSync('photo.png')]), 'photo.png');
formData.append('format', 'WEBP');

fetch('http://localhost:8000/api/v1/convert/image', {
  method: 'POST',
  headers: { 'X-API-Key': 'gv_your_api_key' },
  body: formData
})
.then(res => res.arrayBuffer())
.then(buffer => fs.writeFileSync('converted.webp', Buffer.from(buffer)));
