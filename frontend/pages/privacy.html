Privacy Policy

Last updated: July 12, 2026. Learn how we handle your uploads and profile credentials with strict security.

1. File Processing Privacy

Any image file you upload to GarutVON's conversion engine is saved in a secure, isolated directory on our local worker cluster. Once the Pillow conversion process completes and compiles your output stream, your original file is instantly purged. Converted files are securely hashed and stored for a maximum of 24 hours to let you download them before automatic purging occurs.

2. Profile and Credential Data

We store registration emails, secure Argon2 password hashes, API metrics, and transaction logs on premium Neon PostgreSQL database instances. We do not sell, distribute, share, or analyze user profile directories under any circumstances.

3. Account Purges

If you choose to delete your account inside your user dashboard, all sessions, active tokens, programmatic keys, conversion histories, and audit data are permanently erased from our live server tables instantly.
