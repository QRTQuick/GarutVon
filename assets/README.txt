Place your GarutVON logo files here for use by the site.

Files to add (recommended names):
- garutvon-logo.png  --> Used as the landing page 3D background and other in-site placements.
- garutvon-favicon.png  --> A 32x32 or 48x48 PNG to be used as the browser tab icon.

After placing the files, the site will automatically reference them at:
 - /assets/garutvon-logo.png
 - /assets/garutvon-favicon.png

If you prefer different names, update the paths in `frontend/css/threedee.css` and `frontend/js/components.js` accordingly.
