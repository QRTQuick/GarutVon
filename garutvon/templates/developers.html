Quick Start

Use your API key to authorize requests. Example endpoints available:

  • POST /api/summarize — Summarize text.
  • POST /api/convert — Convert documents (PDF, DOCX, HTML).
  • GET /api/version — Desktop update and version metadata.
  • GET /api/health — Service health check.

Python

curl -X POST /api/summarize -H "Authorization: Bearer YOUR_KEY" -H "Content-Type: application/json" -d '{"text":"Hello world"}'

Or use the bundled developer client at /dev/main.py which contains a ready-to-run Python/Tkinter client.

JavaScript

fetch('/api/summarize',{method:'POST',headers:{'Authorization':'Bearer YOUR_KEY','Content-Type':'application/json'},body:JSON.stringify({text:'Hello'})})

Other SDKs

Official SDK examples for PHP, Go, Java and C# will be added here. For now, use direct HTTP calls with your key.
